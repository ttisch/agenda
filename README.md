## Development

### Prerequisites

- Node.js (LTS)
- Rust
- Yarn

### Setup

```bash
# Clone the repository
git clone https://github.com/tisch-systems/agenda.git
cd agenda

# Install dependencies
yarn install

# Start the development server
yarn tauri dev
```

### Building

```bash
# Build for production
yarn tauri build
```
